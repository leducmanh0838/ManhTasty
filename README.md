**ManhTasty** is a **recipe sharing app** built with **Django (backend)** and **ReactJS (frontend)**.  
A comprehensive Recipe Sharing Platform designed to connect food enthusiasts. The system allows users to seamlessly post and manage their own recipes. It features an advanced search engine with tag-based filtering (e.g., Vegan, Gluten-free) to help users find exactly what they need. Additionally, an interactive rating and commenting system fosters community engagement and feedback.
